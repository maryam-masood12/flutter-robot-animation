# 🤖 Flutter Robot Animation

An interactive Flutter animation project featuring a robot with smooth UI transitions and custom animations.

## ✨ Features

- Interactive robot animation
- Floating robot effect
- Tap interactions using GestureDetector
- Smooth background and container color animations
- Sliding shape selection panel
- Shape switching (Circle, Square & Triangle)
- 360° rotation animation
- Built using Flutter AnimationController

## 🛠 Built With

- Flutter
- Dart
- AnimationController
- AnimatedBuilder
- GestureDetector
- Tween Animation
- SlideTransition

## 💡 What I Learned

During this project, I practiced:

- Explicit Flutter animations
- Managing multiple AnimationControllers
- Tween animations
- Gesture-based interactions
- UI state management
- Building interactive user interfaces
